হোমপেজ ব্যানার / অফার

হোমপেজের স্লাইডার ব্যানার এখান থেকেই বদলান। ছবির লিংক (URL) দিন, অথবা ছবি ছাড়া শুধু অফারের লেখা দিন। সেভ করলেই সাইটে লাইভ হবে।

কুপন কোড ম্যানেজমেন্ট

নতুন কুপন যোগ করুন — সেভ করলেই সাইটে কাজ করবে। ৩ ধরন: % ছাড় / ৳ টাকা ছাড় / ফ্রি ডেলিভারি। (বিল্ট-ইন FIRSTORDER — ৫%, এক ফোনে একবার — সবসময় চালু।)

➕ নতুন কুপন

পাঠকদের ব্লগ (মাসিক প্রতিযোগিতা)

কাস্টমারদের জমা দেওয়া লেখা এখানে আসবে। পড়ে পছন্দ হলে "প্রকাশ করুন" চাপুন — সাইটের ব্লগ পেজে লাইভ হবে। লেখকের ফোন নম্বরে যোগাযোগ করে 🎁 হাদিয়া পাঠাবেন।

🏆 এ মাসের প্রতিযোগিতার বিষয়
সেভ করলেই ব্লগ পেজে নতুন বিষয় দেখাবে। খালি রেখে সেভ করলে বিষয় ছাড়া সাধারণ আহ্বান দেখাবে।
📥 জমা পড়া লেখা
✅ প্রকাশিত লেখা

ছবি আপলোড (imgbb API)

একবার imgbb-র ফ্রি API key বসিয়ে সেভ করলে বই/ব্যানারে সরাসরি ছবি আপলোড করা যাবে — আর লিংক কপি-পেস্ট লাগবে না। key নিন: imgbb.com → About → API।

Facebook Pixel (বিজ্ঞাপন)

Facebook/Instagram বিজ্ঞাপন চালাতে চাইলে Meta Events Manager থেকে Pixel ID কপি করে এখানে বসান। সেভ করলেই সাইটের সব পেজে চালু হবে। খালি রেখে সেভ করলে বন্ধ হবে।

মডারেটর ম্যানেজ

অনুমোদন দিলে মডারেটর নিজের ইমেইল-পাসওয়ার্ড দিয়ে এই প্যানেলে ঢুকে সব মেইনটেইন করতে পারবে। শুধু Courier API আর মডারেটর ম্যানেজ আপনার হাতে থাকবে।
